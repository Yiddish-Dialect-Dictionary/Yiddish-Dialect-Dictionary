בלוי

'blue'

ETYMOLOGY
===========
{Eastern Yiddish בלאָ from MHG blâ. Schaechter writes that בלוי is borrowed from NHG.}
ON blár 'blue, livid, black', OE blāw, (NE blue is from French!), Scots blae 'blackish; livid, pale', Dutch blauw, OHG blāo 'blue, dark, grey', MHG blā, (gen.) blāwes, ENHG blau, blaw, blo, NHG blau < PGmc. *blēwa- 'blue' < *bʰléh₁-uo- (EUR) < *bʰléh₁-u-s, gen. bʰlh₁-uó-s; cf. Latin flāvus 'blond', Old Irish blá 'yellow', Welsh blaw 'grey'
