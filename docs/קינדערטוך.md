קינדערטוך

WESTERN
========

Kindertuch {HOLLAND, Beem, Jerosche}
	•	Chotsche in e beschissen (kinder) tuch. 'Even if it was in a decent (children's) cloth. I want to have it in whatever state it may be.' {Jerosche №469}
