‘tooth’

צאָן

'tooth'

ETYMOLOGY

Got. tunþus✱, ON tǫnn, OE tōð, OS tand, OHG zan(t) < Germ. tanþ- ~ ✱tunþ-; from PIE ✱h₃dont-, from the root ✱h₃ed- 'to bite, be sharp'; cf. Latin dēns, Sanskrit dān, Lithuanian dantìs
