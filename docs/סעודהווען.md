סעודהווען
'to revel, feast'

ETYMOLOGY
===========
A Semitic root (see סעודה) with the Slavic suffixe -eve-, perhaps inspired by Polish biesiadować 'to feast'. The verbal suffix ־עווע־ is derived from the Polish suffixes -owa-, -ewa-, -ywa-, -iwa-.
[Geller 2023: 116]
