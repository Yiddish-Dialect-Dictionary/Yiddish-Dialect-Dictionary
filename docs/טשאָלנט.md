דער/דאָס

ETYMOLOGY
===========
Loez < Latin calentem (accusative present participle) 'that which makes warm'. The /č/ in the East Yiddish dialects represents the original Loez/Old French pronunciation.
{M. Weinreich, Geshikhte 2: 55}
