הייסן
געהייסן
'to be called; to command'

ETYMOLOGY
===========
Got haitan, ON heita, OE hātan, OS hētan, OHG heiʒan, MHG heiʒen, NHG heissen < PGmc. *haitan- 'to call' < PIE *k̂oid-, from the root *k̂ei-/k̂i-; cf. Latin ciēre 'to move, put in motion', citus 'fast'
{EWA 4, 916, Feist 236, Kroonen 2013: 202}
