לײַדן
געליטן
'to suffer'

ETYMOLOGY
===========
Got. ga-leiþan 'to go, travel', ON līða, OE līðan, OS līthan, OHG līdan 'to go, pass, tolerate, suffer', MHG līden 'to suffer', NHG leiden 'to suffer' < PGmc. *līþan- 'to go (by), pass' < PIE *léit-e- / *ley + -t-; cf. Greek loitē 'tomb', Latin lītus 'sea-shore'
{Bammesberger 1990: 56; EWA 5, 1247, Feist 8, Kroonen 2013: 340}
